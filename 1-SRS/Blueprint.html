🔄 State Machine — วงจรชีวิต AI Orb 🔄 State Machine — AI Orb Lifecycle 

stateDiagram-v2
    [*] --> idle
    idle --> listening: user clicks mic / voice wake
    listening --> thinking: speech captured
    thinking --> speaking: LLM response ready
    speaking --> idle: TTS complete
    thinking --> error: API fail / network error
    speaking --> error: TTS error
    error --> idle: user dismisses
    speaking --> listening: user interrupts (continuous mode)

📞 Sequence — Happy Path (ถาม-ตอบปกติ) 📞 Sequence — Happy Path (Normal Q&A) 

sequenceDiagram
    actor U as User
    participant UI as Aether UI
    participant STT as Speech Recognition
    participant Chat as Chat Manager
    participant LLM as DeepSeek API
    participant TTS as Speech Synthesis

    U->>UI: Click mic button
    UI->>STT: start()
    STT-->>UI: onresult(transcript)
    UI->>Chat: addMessage(user, transcript)
    Chat->>LLM: POST messages[]
    LLM-->>Chat: stream tokens
    Chat->>UI: updateMessage(token by token)
    LLM-->>Chat: [DONE]
    Chat->>TTS: speak(response)
    TTS-->>U: audio output
    Chat->>UI: save to localStorage
